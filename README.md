# BikeEasy Mini Program

## Project Overview
BikeEasy is a mini program designed to facilitate bicycle rentals and tracking for users. With an intuitive interface and powerful features, it aims to provide a seamless experience for both users and administrators.

## Features
- **User Registration and Login:** Secure sign-up and login mechanism.
- **Bicycle Availability:** Check real-time availability of bicycles in various locations.
- **Rental Management:** Easy rental process and management for users.
- **Tracking:** Monitor rental status and duration.
- **Admin Dashboard:** Manage users, bicycles, and rentals.

## Structure
- **src/**: Contains all source code files including components, pages, and assets.
- **pages/**: Contains all the pages of the mini program, such as login, registration, rental status, etc.
- **components/**: Reusable components used across different pages.
- **assets/**: Images, icons, and other assets used in the application.
- **utils/**: Utility functions and helpers.

## Installation Steps
1. Clone the repository:  
   `git clone https://github.com/youar10/bikeEasy-mini-program.git`
2. Navigate to the project directory:  
   `cd bikeEasy-mini-program`
3. Install dependencies:  
   `npm install`
4. Run the application:  
   `npm start`

## Page Descriptions
- **Home Page:** Displays the available bicycles and allows users to navigate to other sections.
- **Login Page:** Users can log in to access their accounts.
- **Registration Page:** New users can create an account.
- **Rental Status Page:** Displays the current rental information for logged-in users.
