# BikeEasy Mini Program

## Project Description
The BikeEasy Mini Program is designed to provide users with an intuitive platform to explore and rent bicycles conveniently. The application aims to connect bike owners with renters while promoting eco-friendly transportation.

## Features
- User Registration and Login
- Bicycle Listing with Availability
- Reservation System
- User Ratings and Reviews
- Integrated Map for Location Services

## Structure
```
/bikeEasy-mini-program
├── /components          # Reusable UI components
├── /pages              # Application pages
│   ├── Home           # Home page
│   ├── Listing        # Bicycle listings
│   ├── Profile        # User profile management
│   └── About          # About the application
├── /utils             # Utility functions
└── app.js             # Main application file
```

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/youar10/bikeEasy-mini-program.git
   ```
2. Navigate to the project directory:
   ```bash
   cd bikeEasy-mini-program
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
